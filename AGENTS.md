# dlt-ledger · 项目交接文档（供 AI 代理阅读）

## 项目是什么

中国体彩大乐透的**个人投注账本 PWA**，部署于 GitHub Pages：
https://zhouguiyanguge-droid.github.io/dlt-ledger/

单用户、纯前端、零依赖、零构建。核心价值：真实记账、自动核对中奖、诚实呈现概率，帮用户理性购彩。

## 文件结构

| 文件 | 说明 |
|---|---|
| `index.html` | 全部应用（HTML+CSS+JS 单文件，约900行），无任何外部依赖 |
| `sw.js` | Service Worker，cache-first 离线缓存，缓存名 `dlt-v1` |
| `manifest.webmanifest` | PWA 清单 |
| `icon-192.png` / `icon-512.png` | 图标 |
| `部署说明.md` | 面向用户的部署文档 |

## 核心架构（index.html 内）

- 状态：单一 `state` 对象 → `localStorage`，键 `dlt_ledger_v1`
  - `bets`: 投注号码数组 `{front:[5个1-35], back:[2个1-12], note:选号依据}`，上限 `MAX_BETS=5`
  - `draws`: 开奖记录 `{period, front, back, cost, totalWin, results[], ts}`，results 内嵌当期 bets 快照（号码后续修改不影响历史）
  - `photos`: 彩票照片 `{period, dataUrl(压缩JPEG), ts}`
  - `budget` / `unitPrice`
- 中奖判定 `judge(fHit,bHit)`：大乐透九等奖规则，一/二等奖浮动（用户手输金额），三至九等固定 {10000,3000,300,200,100,15,5}
- 概率引擎 `tierProbs()`：组合数学精确计算各奖级单注概率（一等奖 1/21,425,712）
- 自带 `ui.confirm/prompt/pick/toast` 弹层（**不要改回原生 alert/confirm**——浏览器会拦截连续弹窗导致按钮失灵，这是修过的 bug）
- `fetchLatest()`：调体彩官方接口拉最新开奖（`webapi.sporttery.cn/.../getHistoryPageListV1.qry?gameNo=85`），失败时降级为手动录入
- 模块：预算止损、号码热门度体检、蒙特卡洛模拟器、运气统计（理论vs实际）、选号方法实验室（周易/斐波那契/机选避热/纯随机）、票据照片存档

## 不可妥协的红线（任何迭代不得违反）

1. **禁止加入任何声称或暗示能提高中奖概率的功能**——冷热号预测、走势分析、AI选号推荐、"智能算法"等全部禁止。彩票每期独立随机，这类功能在数学上是欺骗。现有"机选避热"只影响中大奖时的奖金平分风险，且界面已如实说明，这个表述边界必须保持。
2. **「诚实概率面板」和理性购彩提示不得删除或弱化**。
3. **每期5注纪律（MAX_BETS=5）不得放开**——这是用户给自己定的预算纪律。
4. **保持零依赖、零构建、单文件应用**——不引框架、不引CDN库、不加打包工具。这是为了让非程序员用户能直接在 GitHub 网页上传更新。
5. **数据只存用户本地**（localStorage），不得加入任何把投注数据上传到服务器的功能。
6. 中文界面，面向非技术用户。

## 已知技术债 / 值得评估的优化方向

- `index.html` 体积渐大，但拆分与红线4冲突，需权衡（可接受现状）
- 用户输入的渲染大多经过 `esc()`，但值得全面审计 XSS 面
- 照片存 localStorage（约5MB上限），多照片会爆——可评估迁移 IndexedDB（注意保持导出/导入兼容）
- sw.js 缓存名固定 `dlt-v1`，index.html 更新后老缓存可能顶着不放——需要版本号策略
- 移动端：iOS safe-area、键盘弹出遮挡、号码输入体验（可做点选式选号盘）
- `fetchLatest` 依赖官方接口 CORS 放行，策略可能变化，注意保持优雅降级
- 无自动化测试：`judge()` 与 `tierProbs()` 是最值得加测试的纯函数
- 历史记录无分页，长期使用（数百期）后的渲染性能
- 备份提醒机制（数据只在本地，清缓存即丢）

## 验证要点（改动后必须自测）

- `judge`：18组边界用例（5+2→1等 … 1+1/2+0/0+0→不中奖）
- `tierProbs` 总中奖概率 ≈ 6.67%，一等奖精确 1/21,425,712
- 弹层流程：删除注、满5注时加入需弹替换选择、录入开奖含一/二等奖时的金额输入
- 离线打开、手机"添加到主屏幕"
