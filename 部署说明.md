# 大乐透投注账本 PRO · 手机App化部署说明

这个文件夹就是完整的网页App（PWA）。部署到任意免费托管平台后，手机浏览器打开网址，"添加到主屏幕"即可像App一样使用（有图标、全屏、可离线打开）。

## 方法一：GitHub Pages（推荐，永久免费）

1. 注册/登录 github.com
2. 右上角 ＋ → New repository，名字随意（如 `dlt`），选 Public，点 Create
3. 进入仓库 → Add file → Upload files → 把本文件夹里的**全部文件**拖进去（index.html、manifest.webmanifest、sw.js、两个图标）→ Commit changes
4. 仓库 Settings → Pages → Branch 选 `main`，目录选 `/ (root)` → Save
5. 等约1分钟，页面顶部会显示你的网址，形如 `https://你的用户名.github.io/dlt/`

## 方法二：Vercel / Netlify（拖拽即可）

注册后选 "Deploy / Add new site" → 直接把整个文件夹拖进去 → 自动生成网址。

## 手机安装成"App"

- **安卓（Chrome/Edge）**：打开网址 → 菜单 → "添加到主屏幕" / "安装应用"
- **iPhone（Safari）**：打开网址 → 分享按钮 → "添加到主屏幕"

## 功能说明

- **⟳ 自动获取开奖**：尝试调用体彩官方接口自动填入最新开奖号。该接口对网页的放行策略可能变化，若提示获取失败，去官方App查开奖号手动录入即可，其余功能不受影响。
- **数据存储**：数据存在每台设备各自的浏览器里（localStorage），**不会自动跨设备同步**。换手机/清缓存前，先用"导出备份"，再在新设备"导入备份"。
- **离线可用**：安装后断网也能打开记账，联网功能（自动获取开奖）需要网络。

## 注意

- 部署后所有人持有网址都能打开页面，但**看不到你的数据**（数据只在你自己设备里）。
- 想要多设备实时同步需要后端数据库和账号系统，属于另一个量级的工程，目前用导出/导入备份最简单可靠。
